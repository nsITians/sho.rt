## How to use?
Make sure you have node and npm installed.
  <br>
  * install mongodb
    <pre> npm install mongodb</pre>
  * install express
      <pre> npm install express</pre>
  * install shortid
      <pre> npm install shortid</pre>
  * install body-parser
      <pre> npm install body-parser</pre>

Finally:
  * run mongodb
      <pre> mongod </pre>
  * run app.js
      <pre> node app.js</pre>
  * open the localhost
      <pre>localhost:5000/short<pre>
